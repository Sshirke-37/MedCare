# 🌟 Medicine Reminder App

The **Medicine Reminder App** is an Android application designed to help elderly individuals and their caregivers manage daily medication routines with ease. As people age, remembering to take medicines on time and keeping track of stock can be challenging. This app provides a simple, accessible, and reliable solution built using **Java, Android Studio, Room Database, and Firebase**.

---

## ✨ Why This App?
- Many elderly users forget doses or struggle with complex apps.
- Caregivers often need visibility into medicine schedules and stock.
- This app bridges the gap with **reminders, inventory tracking, and caregiver alerts** in one place.

---

## 📱 Key Features
- 🔔 **Medicine Reminders**: Scheduled notifications so users never miss a dose.
- 🗣️ **Voice Alerts**: Text‑to‑Speech reminders for users with poor eyesight.
- 📦 **Inventory Tracking**: Monitor medicine stock and get low‑inventory alerts.
- 👨‍👩‍👧 **Caregiver Notifications**: Automatic alerts via Firebase Cloud Messaging.
- 🚨 **Emergency SOS Button**: Quick alerts to caregivers in urgent situations.
- 📊 **Adherence Reports**: Track dose history and generate reports for doctors or family.

---

## 🛠️ Tech Stack
- **Frontend**: Java, XML (Android Studio)
- **Database**: Room / SQLite
- **Backend**: Firebase (Cloud Messaging, Firestore)
- **Optional APIs**: Twilio (SMS), Google Fit (health data)

---

## 🚀 Getting Started

### Prerequisites
- Install **Android Studio** and **Java JDK**.
- Set up a **Firebase project** (for notifications and data sync).
